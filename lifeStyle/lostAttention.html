    # 克服自己经常忘东西的习惯

    # 案例描述

    1. 之前在出地铁站之后 把钱包放在共享单车的车座上 然后脑子里闪出别的念头 又不准备骑单车了 然后钱包放在车座上忘记拿了

    2. 去健身房练腹的时候 双腿抬起 卡滑下来 丢失在瑜伽垫上

    # 如何规避

      • 不带钱包
      • 脑子不清醒 不出门
      • 每隔半小时检查一下口袋
      • 把贵重物品及卡片放在临时储物柜里
      • 离开场所的时候 仔细检查自己的口袋
      • 穿口袋比较紧实的裤子

    # 如何解决

    1. 保持冷静 迅速筛选出几个可能发生遗漏的具体位置 找到相关的人进行询问
    2. 卡片及时挂失
    3. 手机用找回功能进行找回

    一系列合理的习惯可以减少丢失的风险，自己的感觉是靠不住的，同时不要责怪自己，事物总是朝着熵増的方向发展，不要等到失去才追悔
